## RstocksPkg

[RstocksPkg](https://github.com/deandevl/RstocksPkg) is an R computer script package that provides easy access to US stock market prices and dividends.  The package includes several demonstrations of its capabilities. A study on [dividend analysis](https://github.com/deandevl/R_Dividends_Analysis_Articles) also incorporates many of the packages functions.

To install *RstocksPkg* enter the following command lines from an R console:

```R
install.packages("devtools")
```

```R
devtools::install_github("deandevl/RstocksPkg")
```

In your R application file include the package:

```R
library(RstocksPkg)
```
